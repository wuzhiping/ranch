## 七诫
《动物农场》中动物们制定的基本法律，旨在保障平等与自由
1.凡用两条腿走路的都是敌人
  * ：将人类视为压迫者，激发动物反抗精神。 
2.凡用四条腿走路或者有翅膀的都是朋友
  * ：强调动物之间的团结与互助。 
3.任何动物不得穿衣服
  * ：象征动物的自然状态，反对人类化。 
4.任何动物不得睡床
  * ：防止模仿人类的奢侈生活，保持简朴。 
5.任何动物不得饮酒
  * ：避免沉溺和腐化，保持清醒。 
6.任何动物不得互相残杀
  * ：维护动物之间的和平与安全。 
7.所有动物皆平等
  *：体现动物主义的核心理念——平等与自由。
